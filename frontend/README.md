Frontend (React + Vite) Template

Overview
- React 18 + React Router + Axios
- Vite for fast dev and build

Getting Started
1. Install Node.js (LTS recommended).
2. Install deps: npm install
3. Copy .env.example to .env and set VITE_API_URL.
4. Run dev server: npm run dev
5. Build for production: npm run build
6. Preview build: npm run preview

Structure
- public/index.html: App shell
- src/index.js: React entry
- src/App.js: Router + basic layout
- src/services/api.js: Axios instance
- src/styles: Global styles
